Для расположения элементов друг над другом можно использовать для каждого из элементов свойство z-index, значение которого определяет место расположения элемента. Чтобы задать дополнительную внешнюю границу можно использовать outline. В отличие от border свойство outline не участвует в блочной модели CSS и не изменяет размер элемента. Поэтому его используют, когда хотят добавить рамку без изменения других CSS-параметров. Свойств outline-top, outline-left не существует. Все современные браузеры также поддерживают свойство outline-offset, задающее отступ outline от внешней границы элемента.
